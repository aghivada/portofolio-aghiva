# 🚀 Personal Portfolio Website - Aghiva

Halo! Ini adalah repositori kode untuk website portofolio pribadi saya. Website ini dirancang sebagai platform interaktif untuk menampilkan profil, keahlian, serta berbagai proyek pengembangan aplikasi yang telah saya kerjakan.

Website ini sudah sepenuhnya dideploy dan dapat diakses secara publik melalui tautan berikut:
🔗 **[portofolio-aghiva.vercel.app](https://portofolio-aghiva.vercel.app)**

---

## 🛠️ Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi *front-end* statis yang ringan, responsif, dan diintegrasikan dengan layanan pihak ketiga:

*   **HTML5** – Untuk menyusun struktur dan konten halaman.
*   **CSS3** – Untuk desain tata letak (*layouting*), animasi interaktif, dan visualisasi UI.
*   **JavaScript (ES6)** – Untuk logika interaktif, manipulasi DOM, dan pengelolaan pengiriman data.
*   **Formspree API** – Digunakan sebagai *backend service* pihak ketiga untuk memproses formulir kontak secara *asynchronous* (AJAX/Fetch) langsung ke email pribadi tanpa *reload* halaman.

---

## ✨ Fitur Utama

1.  **Desain Responsif & Modern:** Tampilan website yang adaptif dan tetap rapi saat diakses melalui perangkat *mobile*, tablet, maupun desktop.
2.  **Panel Galeri Proyek:** Menampilkan daftar *showcase* proyek aplikasi yang telah diselesaikan.
3.  **Formulir Kontak Integrasi Email:** Fitur *Contact Me* interaktif yang memungkinkan pengunjung atau rekruter mengirimkan pesan langsung ke kotak masuk email saya tanpa mengalami kendala *error* 405 atau pengalihan halaman (*silent submit*).
